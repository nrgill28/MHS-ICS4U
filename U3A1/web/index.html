Student Manager

Getting Started:

  1. Click the Students List tab
  2. Click Add new
  3. Enter the student details
  4. Click save!

That's it!
A couple of things to note: the library I'm using to make this a desktop app currently has an issue with switching the pages too fast. If the program just dies randomly, that's why. To avoid this, I enforce a 1 second delay on all the links, so if a button doesn't work the first time, give it another go after a second.
